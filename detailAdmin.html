Wellet Admin

I collaborated in developing a dashboard admin linked to the PWA from scratch, aiming to provide necessary information for decision-making, user and profile management, graphs and metrics, as well as forms for creating events, products, or services. It also included informative tables on reservation statuses, commissions, and financial movements, payment release, comment management, and conflict handling. Filtering and data manipulation tools were implemented, along with report generation features, among others. Code architectures were designed to ensure the coexistence of multiple users and secure information editing. React Hook Forms for form control, React Query for data manipulation and API communication, Date FNS for date handling, Swiper JS, Redux, and Context for non-local state management were utilized in the development process.

This project was entirely directed by myself and a colleague, taking responsibility for the entire project architecture with a focus on scalability, reusability, ease of editing, and application.
